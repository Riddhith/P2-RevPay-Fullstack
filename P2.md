Application Overview



RevPay is a full-stack monolithic financial web application that enables secure digital payments and money management for both personal and business users. Personal users can send/request money, manage payment methods, and track transactions through an intuitive web interface. Business users gain additional capabilities including loan applications, payment acceptance, invoice management, and business analytics. The application features robust security measures, role-based access control, real-time transaction processing, and comprehensive notification systems.



 



Core Functional Requirements



As a Personal Account User, I should be able to:



Register and create a personal account with full name, email, phone number, password, and security questions

Login to my account using email/phone and password

View account dashboard with wallet balance, recent transactions, and quick actions

Send money to other users by username, email, phone number, or account ID with amount and optional note

Request money from other users with amount and purpose

View incoming money requests (pending, accepted, declined)

Accept or decline incoming money requests

View outgoing money requests and cancel pending requests

Add payment methods (credit/debit cards) with card number, expiry, CVV, and billing address

Manage payment methods (view all, edit, delete, set default)

View comprehensive transaction history with all transactions

Filter transactions by type (sent, received, added funds, withdrawal), date range, amount, and status

Search transactions by recipient/sender name or transaction ID

Export transaction history to CSV/PDF

Add money to wallet from linked cards (simulated)

Withdraw money from wallet to bank account (simulated)

Receive in-app notifications for transactions, money requests, card changes, and low balance alerts

Manage notification preferences (enable/disable by type)

View notification history with read/unread status

Update profile information

Change password with current password verification

Set/change transaction PIN

As a Business Account User, I should be able to:



Note: Business accounts have all personal account features plus the following:



Register as a business account with business name, business type, tax ID, business address, and contact information

Submit verification documents for business account approval (simulated)

Add business payment methods including business credit/debit cards and bank accounts

Accept payments from customers by generating payment requests

Process incoming payments via customer email, phone, or account ID

Create invoices with itemized line items (description, quantity, unit price, tax)

Add customer information to invoices (name, email, address)

Set payment terms and due dates for invoices

View all invoices with status (draft, sent, paid, overdue, cancelled)

Send invoice notifications to customers (in-app and email)

Mark invoices as paid manually

Track paid and unpaid invoices

Apply for business loans with loan amount, purpose, tenure, and financial information

Upload supporting documents for loan applications

View loan application status (pending, approved, rejected)

View loan details (amount, interest rate, tenure, EMI, repayment schedule)

Make loan repayments (simulated)

View business analytics dashboard with key metrics

View transaction summaries (total received, total sent, pending amounts)

View revenue reports (daily, weekly, monthly)

View outstanding invoices summary

View payment trends with charts

View top customers by transaction volume

Manage business profile information
